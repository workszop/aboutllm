Chatbot vs Agent

Chatbot

  • Reaguje na pytania
  • Jeden obrót konwersacji
  • Brak dostępu do narzędzi
  • Odpowiada tekstem
  • Pasywny

Agent AI

  • Realizuje cele
  • Wielokrotne iteracje
  • Używa narzędzi (API, pliki, bazy)
  • Wykonuje akcje
  • Proaktywny

Elementy agenta

Planer: Dekomponuje zadanie na kroki, decyduje o kolejności działań.

Pamięć: Krótkoterminowa (kontekst) i długoterminowa (baza wiedzy, historia).

Narzędzia: API, dostęp do plików, bazy danych, wyszukiwarki, wykonywanie kodu.

Przykłady: Claude Code, GitHub Copilot Workspace, AutoGPT, Devin
